## ⚡ HIZLI BAŞLANGIÇ

### 📋 **Gereksinimler**
- Python 3.10+
- 8GB+ RAM (önerilen)
- SSD Depolama
- Stabil internet bağlantısı

### 🔧 **Kurulum**

```bash
# 1. Repository'yi klonla
git clone https://github.com/your-repo/proje-phoenix.git
cd proje-phoenix

# 2. Virtual environment oluştur
python -m venv venv
source venv/bin/activate  # Linux/Mac
# veya
venv\Scripts\activate     # Windows

# 3. Bağımlılıkları yükle
pip install -r requirements.txt

# 4. Konfigürasyon dosyası oluştur
cp .env.example .env
# .env dosyasını düzenle (API anahtarları, ayarlar)

# 5. Sistem doğrulaması çalıştır
python scripts/validate_system.py --full-validation
```

### 🚀 **İlk Çalıştırma**

```bash
# 1. Sistem durumunu kontrol et
python main.py status --detailed

# 2. Backtest çalıştır
python main.py backtest --strategy momentum --start-date 2024-01-01 --end-date 2024-03-31 --capital 10000

# 3. Optimizasyon çalıştır
python main.py optimize --strategy momentum --trials 1000

# 4. Canlı trading (dikkatli!)
python main.py live --strategy momentum --capital 1000 --symbol BTC/USDT
```
